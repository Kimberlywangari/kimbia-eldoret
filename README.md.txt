Kimbia Eldoret | Running Club Website
A professional, mobile-responsive community website built for Kimbia Eldoret, a vibrant running and walking club based in Eldoret, Kenya. 
This project focuses on promoting holistic health and community movement.# kimbia-eldoret
View the live site here:
Features
Hero Video Background: High-impact visual introduction to the "Tribe".

Mobile-First Design: Fully responsive navigation and layout for runners on the go.

Interactive FAQ: Custom JavaScript-powered accordion for community questions.

Live Stats Counter: Animated counters showing total runners and weeks of community.

Community Carousel: A seamless, infinite-loop image gallery showcasing the club's activities.

Third-Party Integrations: * Google Calendar: Live schedule for weekly runs at Rupa's Mall.

Social Connectivity: Direct links to the club's WhatsApp community and Strava club profile.

 Tech Stack
HTML5: Semantic structure for SEO and accessibility.

CSS3: Custom styling featuring Flexbox, Grid, and smooth scroll animations.

JavaScript : Intersection Observer API for scroll-reveal effects and custom UI logic.

Font Awesome: For professional social media iconography.
